JOGO:

Usar a estrela para colidir com o cogumelo, esfera e o cubo.

Interações:

Botão_P: congela o jogo (usado para debug)
Botão_Space: simula as colisões (usado para debug)
Botão_0: seta para (0,0,0) a movimentação do personagem. (estrela)

Para se mover nos eixos x,y,z:
Botão_1
Botão_2
Botão_UP
Botão_Left
Botão_Right
Botão_Down
